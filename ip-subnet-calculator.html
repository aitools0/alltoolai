How to Calculate Subnet from IP Address

Many users ask how to calculate subnet from IP address or how to subnet mask calculation works. Here is the simplified process used by our calculator:

  1. Identify the IP and CIDR: Start with an IP (e.g., 192.168.1.10) and a prefix (e.g., /24).
  2. Convert to Binary: Transform both the IP address and the Subnet Mask into binary format.
  3. Perform Bitwise AND: Compare the IP binary and Mask binary. If both bits are 1, the result is 1. Otherwise, it is 0.
  4. Result: The output of the AND operation is your Network Address. The inverted bits give you the Wildcard Mask.
